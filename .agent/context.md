# Контекст проекта: Epic Free Games Bot

## 1. Общее описание
Проект представляет собой телеграм-бота для канала **GrampsGamer**.  
Бот предназначен для автоматического информирования подписчиков о еженедельных бесплатных раздачах игр в магазине Epic Games Store.

## 2. Ссылки на проект
- **Telegram-бот:** [@gamefreebie_bot](https://t.me/gamefreebie_bot)
- **Telegram-канал:** [@grampsgamer](https://t.me/grampsgamer)

## 3. Цели и задачи
- **Основная задача:** Получение данных о бесплатных играх через публичный API Epic Games Store.
- **Вторичная задача:** Формирование и отправка сообщений в Telegram-канал с описанием игр, датами раздачи, ссылками и изображениями-обложками.
- **Формат публикаций:** Если игра одна — одиночный пост с обложкой. Если игр несколько (обычно две) — они собираются в одно сообщение в виде альбома (Media Group) с общим текстом-описанием под первым фото. Разметка текста: MarkdownV2.
- **Целевая аудитория:** Подписчики канала GrampsGamer (геймеры, интересующиеся инди-разработкой и геймдизайном).

## 4. Технические детали реализации
- **API Endpoint:** `https://store-site-backend-static.ak.epicgames.com/freeGamesPromotions`
- **Язык разработки:** Python версии 3.13 (64-bit)
- **Среда разработки (IDE):** Visual Studio Code версии 1.125.0
- **ОС разработчика:** Windows 11 Pro (64-bit)
- **Характеристики машины:** 13th Gen Intel Core i7-13620H (2.40 GHz), RAM 32 ГБ.
- **Режим работы:** Постоянно запущенный демон (Вариант А) с периодической проверкой раздач 1 раз в день (время проверки настраивается в конфигурации).
- **Ручное управление:** Команда `/check` в личных сообщениях бота, доступная только владельцу и администраторам канала. Она принудительно проверяет игры прямо сейчас и отправляет превью администратору (в ЛС) или публикует их в канал (команда `/check channel`).
- **Локализация:** Текст публикаций формируется на русском языке (если доступен в API Epic Games Store, иначе используется английский язык).
- **Хранение истории отправленных раздач:** Локальный JSON-файл для предотвращения дублирования постов при фоновой автоматической проверке.
- **Управление зависимостями:** Использование стандартного `requirements.txt` внутри виртуального окружения Python (`.venv`).

## 5. О проекте и авторе (GrampsGamer)
Канал **GrampsGamer** посвящен играм и геймдеву. В нем автор освещает:
- Основы геймдизайна.
- Личный опыт разработки небольших пет-проектов (игр).
- Участие в игровых джемах (Game Jams).
- Истории и инсайты из мира инди-разработки.

Данный бот является инструментом для улучшения взаимодействия с сообществом канала.
