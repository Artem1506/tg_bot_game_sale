# Навыки и инструменты проекта (Skills & Stack)

## Основной стек (Tech Stack)
- **Язык:** Python 3.13
- **HTTP клиент:** `aiohttp` (для асинхронных запросов к API Epic Games).
- **Telegram Bot:** `aiogram` версии 3.x (асинхронная библиотека для Telegram Bot API).
- **Переменные окружения:** `python-dotenv`.
- **База данных:** JSON-файл в корневого каталоге проекта для сохранения истории отправленных раздач.
- **Зависимости:** Стандартный `requirements.txt`.

## Работа с API Epic Games
- Использование REST API (`GET` запросы к Endpoint).
- Асинхронная обработка JSON-ответов.
- Фильтрация данных: извлечение названия игры, описания, дат старта и окончания раздачи, ссылок на изображения-обложки.
- Поддержка локализации: запрос к API с параметром `locale=ru-RU` и автоматическим фолбеком на английский язык, если русское описание отсутствует.

## Шаблоны проектирования (Design Patterns)
- **Singleton:** Для сессии HTTP-клиента и менеджера конфигурации.
- **Factory:** Для генерации текстовых Markdown-сообщений под отправку в канал.
- **Repository / Service:** Для изоляции логики API (Epic Games Client) от логики отправки в Telegram.
- **Media Group Wrapper:** Логика группировки нескольких изображений в один альбом через `InputMediaPhoto` с привязкой текста к первому элементу.

## Дополнительные утилиты
- **Планировщик задач:** Использование встроенного механизма `asyncio` (запуск бесконечного цикла, проверяющего время раз в минуту, либо библиотеки `apscheduler`) для вызова проверки раз в день в настроенное время (например, в 19:00 ежедневно).
- **Форматирование дат:** Библиотека `datetime` (обработка временных меток в UTC и локальном времени).
